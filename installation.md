# Installation

## Synopsis

Notes on installing `tsd-file-api` on various platforms.

## RHEL/CentOS 8

Besides

* git
* python3-dev

the following packages are required:

* postgresql-server
* libpq-dev
* libpq-devel.x86_64
* a NaCl-derivative, like `libsodium`
* optionally `sqlite-amalgamation` (see file `custom-sqlite-for-rhel7.md`)
